# Changelog

All notable changes to Santri Analytics will be documented in this file.

## v0.1.0-alpha

### Added

- Initial public alpha release
- Student management module
- Memorization tracking module
- Attendance logging
- Behavior/incident records
- Parent account support
- User management
- Monthly leave feature
- Dashboard and reporting views
- Google Apps Script + Google Sheets backend

### Notes

This is an early alpha version. The project is actively being refactored, documented, and prepared for wider open-source collaboration.
