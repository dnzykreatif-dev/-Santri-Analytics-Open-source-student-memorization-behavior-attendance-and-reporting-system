# Code of Conduct

Santri Analytics is an open-source project intended to support education communities.

## Our Standards

We expect contributors to:

- Be respectful
- Use constructive language
- Help others learn
- Avoid insults and personal attacks
- Focus on improving the project

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment
- Discrimination
- Personal attacks
- Publishing private information
- Intentionally harmful contributions

## Enforcement

The maintainer may remove comments, issues, or pull requests that violate this code of conduct.
