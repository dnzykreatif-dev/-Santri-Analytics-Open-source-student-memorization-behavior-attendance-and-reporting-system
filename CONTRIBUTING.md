# Contributing to Santri Analytics

Thank you for your interest in contributing to Santri Analytics.

This project is built to help TPQ, pesantren, and small Islamic education institutions manage students, memorization, attendance, behavior records, and reports using Google Apps Script and Google Sheets.

## How to Contribute

You can contribute by:

- Reporting bugs
- Suggesting features
- Improving documentation
- Testing the app
- Refactoring code
- Improving UI/UX
- Reviewing security and data handling

## Development Workflow

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Submit a pull request.
