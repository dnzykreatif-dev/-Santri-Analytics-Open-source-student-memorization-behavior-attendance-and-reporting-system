
# Roadmap

Santri Analytics is currently in early alpha. This roadmap explains the direction of the project and helps contributors understand what will be improved next.

## Current Status

- Project status: Alpha
- Main stack: Google Apps Script + Google Sheets
- Target users: TPQ, pesantren, and small Islamic education institutions
- Main goal: Provide a low-cost student management system without paid server infrastructure

## Short-Term Priorities

- Improve mobile responsive layout
- Fix bottom navigation overlapping content
- Stabilize student, attendance, memorization, and behavior modules
- Improve documentation for installation and setup
- Add sample Google Sheet schema/template
- Add screenshots and usage guide

## Medium-Term Priorities

- Improve role-based access control
- Add better parent account experience
- Add exportable reports
- Improve dashboard analytics
- Add safer data validation
- Refactor frontend modules for maintainability

## Long-Term Vision

Santri Analytics aims to become an open-source management system for Islamic education institutions that need affordable, practical, and maintainable digital tools.
