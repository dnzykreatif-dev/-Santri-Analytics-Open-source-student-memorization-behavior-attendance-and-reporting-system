# Security Policy

Santri Analytics is currently in alpha. Security improvements are part of the active roadmap.

## Reporting a Vulnerability

If you find a security issue, please do not publish it publicly in an issue.

Please contact the maintainer privately or open a minimal issue saying that a security concern exists without exposing sensitive details.

## Security Areas of Interest

This project especially needs review in:

- Google Apps Script permissions
- Google Sheets access control
- User authentication
- Parent account access
- Sensitive student data handling
- Input validation
- Client-side exposure of private data

## Supported Versions

Currently, only the latest alpha version is supported.

## Maintainer Response

The maintainer will review reported security concerns and prioritize fixes based on risk and impact.
